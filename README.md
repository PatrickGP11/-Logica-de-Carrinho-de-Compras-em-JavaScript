# 🛒 Lógica de Carrinho de Compras em JavaScript

Este projeto faz parte do meu treinamento intensivo de aprimoramento em JavaScript e desenvolvimento de software. O objetivo desta aplicação foi construir a lógica de processamento de um carrinho de compras de e-commerce utilizando JavaScript puro (Vanilla JS).

## 🚀 Tecnologias e Conceitos Aplicados

- **JavaScript (ES6+)**
- **Node.js** para execução local no terminal
- **Manipulação de Arrays e Objetos**
- **Higher-Order Functions:**
  - `map()`: Utilizado para extrair e criar novas listas com dados específicos (ex: apenas nomes dos produtos).
  - `filter()`: Utilizado para aplicar regras de negócio e filtrar produtos por categoria.
  - `reduce()`: Utilizado para processar cálculos complexos, como a soma do valor total do pedido multiplicando preço x quantidade.

## ⚙️ Como executar o projeto

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.
2. Clone este repositório.
3. Abra o terminal na pasta do projeto e execute o comando:
   ```bash
   node carrinho.js
